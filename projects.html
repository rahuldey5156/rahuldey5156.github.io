Projects
> 01. Single-Cell RNA-seq Analysis of Mouse Embryonic Stem Cells
[ GitHub ]
Reproduced and extended a published scRNA-seq study using 869 mouse embryonic stem cells across 3 conditions. Built end-to-end pipelines in both R and Python, comparing scran, Seurat, and Scanpy methodologies. Identified a rare 2C-like totipotent subpopulation via Zscan4 family gene expression.
tools: R · Seurat · Bioconductor · Python · Scanpy · Kallisto · FastQC · Git
> 02. MTR4-Mediated RNA Surveillance in Mouse Spermatogenesis
[ GitHub ]
Analysed a large-scale RNA-seq dataset (GSE253154) to investigate MTR4's role in RNA surveillance during spermatogonial differentiation. Performed differential expression with DESeq2, pathway enrichment with fgsea, and exploratory PCA and clustering to detect transcriptomic differences between conditions.
tools: R · DESeq2 · fgsea · STAR · featureCounts · BiomaRt · MSigDB · Git
> 03. ProtExplorer — Web-Based Protein Sequence Analysis Tool
[ GitHub ] [ Live ]
Developed a full-stack bioinformatics web platform for automated comparative protein sequence analysis. Implemented a six-stage pipeline integrating NCBI Entrez, Clustal Omega alignment, PROSITE motif scanning, phylogenetic tree construction, and AlphaFold structure lookup. Built an interactive phylogenetic tree visualisation using D3.js with zoom, pan, and node collapse.
tools: PHP · MySQL · Python · BioPython · Clustal Omega · D3.js · EMBOSS · Git
